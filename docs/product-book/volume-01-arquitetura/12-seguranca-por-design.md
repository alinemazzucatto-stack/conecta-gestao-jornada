# Segurança por Design

| Área | Controle |
|---|---|
| Identidade | MFA, SSO e recuperação segura |
| Sessão | Expiração e revogação |
| Autorização | RBAC com escopo |
| Dados | Criptografia e mascaramento |
| APIs | JWT/OAuth2 e rate limit |
| Evidências | Hash e armazenamento seguro |
| Auditoria | Log imutável |
| Integrações | Segredos protegidos |
| Aplicação | Validação e proteção contra ataques |

## LGPD

- finalidade definida;
- coleta mínima;
- retenção configurável;
- trilha de acesso;
- mascaramento em ambientes de teste;
- anonimização ou exclusão conforme política.
